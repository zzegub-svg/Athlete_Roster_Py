# Athlete_Roster_Py

UT Austin Athlete Database Management System

A Python console program for managing UT Austin athletic teams and athletes.

Classes:

Athlete – Stores athlete info (name, position, year, number) and test results.

Team – Manages a roster of athletes, allows adding athletes, comparing athletes, computing test averages, and loading/saving CSVs.

League – Manages multiple teams, allows creating and switching active teams.

Features:

Create and switch between multiple teams.

Add athletes and record their test results.

Compare athletes’ test performance.

Compute team averages for specific tests.

Load and save rosters to CSV.

Display full team rosters with detailed info.

Usage:

Run athlete_database.py with Python 3:

python athlete_database.py

Follow the menu to manage teams, athletes, and test data.

Dependencies:

Python 3.x

Built-in csv and statistics modules only.
